# Idea del Proyecto

## Nombre del proyecto

Travel Planner

## Descripción

Travel Planner es una aplicación web diseñada para ayudar a los usuarios a organizar y gestionar sus viajes de forma sencilla. La aplicación permitirá registrar destinos, fechas, presupuestos y notas relacionadas con cada viaje.

## Problema que intenta resolver

Muchas personas utilizan notas, documentos o aplicaciones diferentes para organizar sus viajes. Esto dificulta tener toda la información centralizada.

Travel Planner pretende ofrecer una única plataforma donde almacenar y consultar toda la información relacionada con los viajes.

## Usuario objetivo

La aplicación está dirigida a personas que viajan con frecuencia y desean planificar sus viajes de forma organizada.

## Funcionalidades principales

* Crear viajes.
* Editar viajes.
* Eliminar viajes.
* Visualizar todos los viajes registrados.
* Consultar detalles de un viaje.
* Mostrar estadísticas básicas sobre los viajes.

## Funcionalidades opcionales

* Filtros por destino.
* Filtros por fecha.
* Presupuesto total invertido.
* Modo oscuro.
* Categorías de viaje.

## Mejoras futuras

* Sistema de autenticación.
* Compartir viajes con otros usuarios.
* Integración con mapas.
* Integración con APIs meteorológicas.
* Integración con buscadores de vuelos y hoteles.
